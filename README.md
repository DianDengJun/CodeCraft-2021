## IDE要求
可以使用 CLion 或 Visual Studio Code

### CLion
确保已安装 MinGW, 并成功设置环境变量

### VS Code
1. 确保已安装 MinGW, 并成功设置环境变量
2. 安装 CMake, 版本 2.8 以上
3. 安装 VSCode 插件: C/C++, CMake, CMake Tools
